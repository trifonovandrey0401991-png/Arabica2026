Политика конфиденциальности

Дата вступления в силу: 1 марта 2026 г.

Приложение «Арабика» (далее — «Приложение») — мобильное приложение сети кофеен «Арабика». Настоящая Политика конфиденциальности описывает, какие данные мы собираем, как их используем и защищаем.

1. Какие данные мы собираем

  • Номер телефона — для входа в приложение и идентификации пользователя
  • Имя и фамилия — для отображения в профиле пользователя
  • Данные о местоположении (GPS) — для отображения ближайших кофеен. Собирается только при использовании соответствующей функции
  • Фотографии — загружаемые пользователем. Фото хранятся на нашем сервере
  • Push-токены — для отправки уведомлений через Firebase Cloud Messaging
  • Данные об устройстве — модель устройства и версия операционной системы для технической поддержки

2. Как мы используем данные

  • Авторизация и аутентификация в приложении
  • Отправка уведомлений
  • Обработка заказов
  • Программа лояльности (баллы, колесо удачи)

3. Хранение данных

Данные хранятся на защищённом сервере, расположенном в Российской Федерации. Доступ к серверу ограничен и защищён шифрованием (HTTPS/TLS). Мы не передаём персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

4. Сторонние сервисы

  • Firebase Cloud Messaging (Google) — для отправки push-уведомлений. Политика конфиденциальности Firebase
  • Telegram Bot API — для отправки одноразовых кодов при сбросе PIN-кода

5. Права пользователя

Вы имеете право:

  • Запросить информацию о хранимых данных
  • Запросить удаление ваших персональных данных
  • Отозвать согласие на обработку данных

Для этого свяжитесь с нами по электронной почте: info@arabica26.ru

6. Геолокация

Приложение запрашивает доступ к геолокации для функций:

  • Карта магазинов — отображение ближайших кофеен

Геолокация используется только при активном использовании приложения и с согласия пользователя. Вы можете отключить доступ к геолокации в настройках устройства.

7. Камера

Приложение запрашивает доступ к камере для:

  • Загрузки фотографий
  • Сканирования QR-кодов (программа лояльности)

8. Безопасность

Мы принимаем разумные меры для защиты ваших данных, включая:

  • Шифрование соединения (HTTPS)
  • Авторизация по PIN-коду
  • Ограничение доступа к серверу по IP
  • Регулярное обновление программного обеспечения

9. Изменения в политике

Мы можем обновлять настоящую Политику конфиденциальности. Актуальная версия всегда доступна по данной ссылке. Рекомендуем периодически проверять эту страницу.
