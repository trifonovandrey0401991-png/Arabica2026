## 2. Критический путь (~15 мин)

> Без этого приложение бесполезно. Проверяй в первую очередь.

- [ ] Открой приложение → видишь экран регистрации

- [ ] Введи имя → нажми «Зарегистрироваться»

- [ ] Попадаешь в главное меню клиента

- [ ] Введи свой PIN → попадаешь в главное меню

### 2.3 Восстановление PIN-кода
- [ ] На экране ввода PIN нажми «Забыли PIN?»

- [ ] Получи код в Telegram-боте → введи его
- [ ] Создай новый PIN → попадаешь в главное меню

### 2.4 Главное меню

- [ ] Видишь кнопки, соответствующие твоей роли

## 10. Порядок проверки (рекомендация)

Если времени мало, проверяй в таком порядке:

1. **Секция 1** — автотесты (запускает разработчик, ~5 мин)

7. **Секция 8** — фоновые процессы (~10 мин)
