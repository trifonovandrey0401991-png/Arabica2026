## 16. КОМАНДЫ АГЕНТОВ

В проекте есть две команды из 6 параллельных агентов (`.claude/commands/`):

**`/team` — для создания нового функционала:**
Когда Андрей просит сделать что-то большое (новый модуль, новая страница + API + тесты). 6 агентов работают параллельно: Фронтенд, Бэкенд, Тестировщик, Архитектор, Сеньор, Безопасник.

**`/team-review` — для проверки существующего модуля:**
Когда нужно проверить качество и безопасность модуля. Те же 6 агентов читают код и ищут проблемы.

**Когда использовать:**
- Задача затрагивает **больше 5 файлов** → `/team`
- Нужно **создать новый модуль** (Flutter + API + тесты) → `/team`
- Нужно **проверить модуль** на баги, безопасность, качество → `/team-review`
- Андрей просит **полную проверку** чего-то → `/team-review`

**Когда НЕ использовать:**
- Маленькие правки (1-3 файла) → делай сам
- Исправление одного бага → делай сам
- Вопрос или анализ без изменений → делай сам

## 17. НАПОМИНАНИЕ
